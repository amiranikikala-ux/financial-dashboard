# DateTimeCalendarPicker вҖ” бғҳбғңбғўбғ”бғ’бғ бғҗбғӘбғҳбғҳбғЎ бғһбғ бғқбғ’бғ бғ”бғЎбғҳ

## бғ бғҗ бғ’бғҗбғҷбғ”бғ—бғ“бғҗ вң…

### 1. бғҷбғқбғӣбғһбғқбғңбғ”бғңбғўбғҳ бғӣбғ–бғҗбғ“бғҗбғҗ
- `src/components/DateTimeCalendarPicker.jsx` вҖ” бғЎбғ бғЈбғҡбғҗбғ“ бғҗбғ¬бғ§бғқбғ‘бғҳбғҡбғҳ
- Props: `fromDate`, `fromTime`, `toDate`, `toTime` + onChange handlers
- бғӨбғЈбғңбғҘбғӘбғҳбғқбғңбғҗбғҡбғҳ: 2-бғ—бғ•бғҳбғҗбғңбғҳ бғҷбғҗбғҡбғ”бғңбғ“бғҗбғ бғҳ, range selection, hover preview, quick presets (бғ“бғҰбғ”бғЎ/бғ’бғЈбғЁбғҳбғң/7бғ“/30бғ“/90бғ“), бғ“бғ бғқбғҳбғЎ бғҗбғ бғ©бғ”бғ•бғҗ 15-бғ¬бғЈбғ—бғҳбғҗбғңбғҳ бғҳбғңбғўбғ”бғ бғ•бғҗбғҡбғҳбғ—, "бғ’бғҗбғӣбғқбғ§бғ”бғңбғ”бғ‘бғҗ" бғҰбғҳбғҡбғҗбғҷбғҳ

### 2. CSS бғЎбғўбғҳбғҡбғ”бғ‘бғҳ бғ“бғҗбғӣбғҗбғўбғ”бғ‘бғЈбғҡбғҳбғҗ
- `src/styles/components.css` вҖ” `dtcp-*` бғҷбғҡбғҗбғЎбғ”бғ‘бғҳ (бғ®бғҗбғ–бғҳ ~1165вҖ“1543)
- бғҗбғ бғЎбғ”бғ‘бғЈбғҡбғҳ `crp-*` бғЎбғўбғҳбғҡбғ”бғ‘бғҳбғЎ бғҗбғңбғҗбғҡбғқбғ’бғҳбғЈбғ бғҳ + бғ“бғҗбғӣбғҗбғўбғ”бғ‘бғҳбғ—бғҳ:
  - `.dtcp-time-row` вҖ” date input + time select row
  - `.dtcp-day--today` вҖ” бғ“бғҰбғ”бғЎ marker (inset box-shadow)
  - Mobile responsive (640px breakpoint)

## бғ бғҗ бғ“бғҗбғ бғ©бғҗ (бғЎбғҗбғӣбғқбғӣбғҗбғ•бғҡбғқ) рҹ“Ӣ

### 4. бғӨбғҳбғҡбғўбғ бғҗбғӘбғҳбғҳбғЎ бғҡбғқбғ’бғҳбғҷбғҗ (optional)
- бғҗбғ бғЎбғ”бғ‘бғЈбғҡбғҳ POS daily table-бғҳбғЎ бғӨбғҳбғҡбғўбғ бғҳ (posDateFrom/posDateTo) бғЁбғ”бғҳбғ«бғҡбғ”бғ‘бғҗ бғ’бғҗбғ“бғҗбғ”бғ‘бғҗбғЎ datetime picker-бғ–бғ”
- бғҗбғң бғӘбғҗбғҡбғҷбғ” бғӨбғҳбғҡбғўбғ бғҗбғӘбғҳбғҗ datetime-бғҳбғ— (бғ‘бғҗбғңбғҷбғҳбғЎ бғўбғ бғҗбғңбғ–бғҗбғҘбғӘбғҳбғ”бғ‘бғҳбғЎбғ—бғ•бғҳбғЎ бғ—бғЈ API-бғЎ бғҗбғҘбғ•бғЎ бғ“бғ бғқбғҳбғЎ бғ•бғ”бғҡбғҳ)

### 5. API integration (бғ—бғЈ бғЎбғҗбғӯбғҳбғ бғқбғҗ)
- `/api/data?tab=cashflow_summary` endpoint-бғЎ бғЁбғ”бғҳбғ«бғҡбғ”бғ‘бғҗ `from`, `to`, `fromTime`, `toTime` query params бғ“бғҗбғ”бғӣбғҗбғўбғқбғЎ

## вң… бғ’бғҗбғҷбғ”бғ—бғ“бғҗ: Cashflow бғҳбғңбғўбғ”бғ’бғ бғҗбғӘбғҳбғҗ (бғңбғҗбғ‘бғҳбғҜбғҳ 3)

### 3. Cashflow бғ’бғ•бғ”бғ бғ“бғ–бғ” бғҳбғңбғўбғ”бғ’бғ бғҗбғӘбғҳбғҗ вҖ” бғ–бғ”бғ“бғҗ бғӣбғ®бғҗбғ бғ”бғЎ вң…
1. **State бғ“бғҗбғӣбғҗбғўбғ”бғ‘бғЈбғҡбғҳбғҗ** `Cashflow.jsx`-бғЁбғҳ (бғ®бғҗбғ–бғҳ ~88вҖ“91):
   - `cashflowFromDate` / `cashflowToDate` вҖ” "YYYY-MM-DD"
   - `cashflowFromTime` / `cashflowToTime` вҖ” "HH:MM" (default 00:00 / 23:59)
2. **Import** вҖ” `DateTimeCalendarPicker` бғ®бғҗбғ–бғҳ 4
3. **JSX** вҖ” tab-hero-бғЎбғҗ бғ“бғҗ kpi-grid-бғЎ бғЁбғқбғ бғҳбғЎ (бғ®бғҗбғ–бғҳ ~348вҖ“358)
4. **бғҳбғңбғҳбғӘбғҳбғҗбғҡбғҳбғ–бғҗбғӘбғҳбғҗ** вҖ” POS daily_summary-бғҳбғЎ бғһбғҳбғ бғ•бғ”бғҡбғҳ/бғ‘бғқбғҡбғқ бғ“бғҰбғҳбғ— (useEffect, бғ®бғҗбғ–бғҳ ~171вҖ“172)
